Configuration

Config lives at ~/.config/dict/config.json. Auto-created on first launch. Edit by hand or use ⌘, from the menu bar.

Recording Speech Engine LLM Modes UI Other Example Related Files

Recording

hotkeyMode string

How the hotkey triggers recording.

"pushToTalk" default

"toggle" — Press ⌥Space to start, press again to stop.

"pushToTalk" — Hold ⌥Space to record, release to stop.

Speech Engine

sttEngine string

Speech-to-text backend.

"apple" default

"apple" — On-device Apple Speech Recognition.

"whisper" — Runs whisper-cli as a subprocess. Requires brew install whisper-cpp + a GGML model.

whisperModelPath string

Absolute path to a Whisper GGML model file. Only used when sttEngine is "whisper".

"" default (empty)

Example: /opt/homebrew/share/whisper-cpp/models/ggml-base.en.bin

whisperLanguage string

Language for speech recognition.

"en" default
"auto" "en" "es" "fr" "de" "it" "pt" "ja" "ko" "zh" "ru" "ar" "hi" "nl" "pl" "sv" "tr" "uk"

LLM Post-Processing

llmEnabled bool

Enable LLM post-processing to clean up transcriptions (fix grammar, remove filler words).

false default
llmProvider string

LLM API provider. Both use their respective chat completions API format.

"openai" default

"openai" — OpenAI-compatible API (also works with Ollama, LM Studio).

"anthropic" — Anthropic Messages API.

llmEndpoint string

API endpoint for LLM requests.

http://localhost:11434/v1/chat/completions default (Ollama)
llmModel string

Model identifier sent to the LLM API.

"llama3.2" default
llmApiKey string

API key for the LLM provider. Leave empty for local endpoints that don't require auth.

"" default (empty)
llmPrompt string

System prompt sent to the LLM for post-processing transcriptions.

Default: "Clean up the following voice transcription. Remove filler words, fix grammar and punctuation. Output ONLY the cleaned text."

Modes

codeMode bool

When enabled, adjusts LLM processing for code-related dictation (preserves technical terms, formatting).

false default
privacyMode bool

Forces offline operation: switches engine to whisper and disables LLM.

false default

UI

verboseOverlay bool

Controls the recording overlay style.

false default

true — Full overlay with mic, audio levels, and streaming transcription.

false — Compact pill with mic and audio levels only.

Other

onboardingDone bool

Tracks whether the onboarding wizard has been completed. Set automatically.

false default
githubRepo string

GitHub repository for update checking.

"" default (empty)

Format: "owner/repo"

Example config

{
  "hotkeyMode": "pushToTalk",
  "sttEngine": "apple",
  "whisperLanguage": "en",
  "whisperModelPath": "",
  "llmEnabled": true,
  "llmProvider": "openai",
  "llmEndpoint": "https://api.openai.com/v1/chat/completions",
  "llmModel": "gpt-4o-mini",
  "llmApiKey": "",
  "llmPrompt": "Clean up the following voice transcription...",
  "codeMode": false,
  "privacyMode": false,
  "verboseOverlay": false,
  "onboardingDone": true
}
